# GreetingSanta
A Santa's doll with leds and a moving arm with a servo and a motion sensor, controlled with Arduino.

Here you can find the Instructable for building the system: https://www.instructables.com/id/Greeting-Santa/

- The system used is an Arduino nano
- A buch of Neopixel leds.
- A servo.
- A motion sensor.
